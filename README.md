## Usage
Here is an example
1. Download `mocha.yaml`
1. Open GlazeWM config file  
`C:\Users\yourusername\.glaze-wm\config.yaml`
1. Add or replace `bar: section`
1. Save and reload GlazeWM `alt+shift+r`
1. Enjoy your new bar!
