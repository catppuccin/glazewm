## Usage

Here is an example

1. copy `mocha.yaml` to `C:\Users\yourusername\.glaze-wm\config.yaml`
2. replace hex color code in `bar:` with variables

```
background: *crust
foreground: *mauve
```

[See example config.yaml](example-config.yaml)
